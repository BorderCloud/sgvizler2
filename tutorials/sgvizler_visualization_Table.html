Definition

Table of results. All columns are treated as text or Headers can be hide.

Table options:

Name Default Description
width 100% Table width
height Table height
headings true true/false Print headers.

Examples

Table example 1 :

  • Result
  • HTML
  • Javascript 
<div id="result"
data-sgvizler-query="
PREFIX rdfs: &lt;http://www.w3.org/2000/01/rdf-schema#&gt;
SELECT  *
WHERE {
        ?object ?property ?valueOrObject .
}
LIMIT 4
"
data-sgvizler-endpoint="https://query.wikidata.org/sparql"
data-sgvizler-chart="sgvizler.visualization.Table"
></div>
