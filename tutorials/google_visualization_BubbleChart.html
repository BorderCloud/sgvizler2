Definition

Line of results.

Line options:

Name Default Description
textStyle Black Controls the color of the bubble's color

Examples

BubbleChart 1 :

With options "textStyle.Color=green"

  • Result
  • HTML
  • Javascript 
<div id="result"
                    data-sgvizler-query="
                    PREFIX bd: <http://www.bigdata.com/rdf#>  
                    PREFIX wikibase: <http://wikiba.se/ontology#> 
                    PREFIX wd: <http://www.wikidata.org/entity/> 
                    PREFIX wdt: <http://www.wikidata.org/prop/direct/> 
                    select  DISTINCT ?paysLabel ?pib ?population ?continentLabel
                    WHERE {
                              ?pays wdt:P31 wd:Q6256.
                              ?pays wdt:P2132 ?pib .
                              ?pays wdt:P1082 ?population .
                              ?pays wdt:P30 ?continent .
                    SERVICE wikibase:label {
                         bd:serviceParam wikibase:language 'en' .
                    }
                    
                          }
                    LIMIT 200" 
                data-sgvizler-endpoint="https://query.wikidata.org/sparql" 
                data-sgvizler-chart="google.visualization.BubbleChart" 
                data-sgvizler-option="sortBubblesBySize:true" 
                data-sgvizler-log="0">
    
></div>
<script src="../browser/sgvizler2.js"></script>
<script>
   //Draw a chart
   //sgvizler2.containerDraw('result');
   //or
   //$("#result").containerchart();

   //Draw all Chart
   sgvizler2.containerDrawAll();
</script>
