Definition

With PivotTable.js, you can explore data and analysis by turning a data set into a summary table and then optionally adding a true 2-d drag'n'drop UI to allow a user to manipulate this summary table, turning it into a pivot table.

Chart options

You can change all the options directly and you can hide all buttons via the switch at the top right. Use the SPARQL editor to create and export options in a new chart.
