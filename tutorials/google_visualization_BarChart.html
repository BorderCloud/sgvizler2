Definition

Line of results.

Line options:

Name Default Description
isStacked false If set to true, stacks the elements for all series at each domain value

Examples

BarChart by default :

With options "isStacked=false" (default) and "sortBarsBySize=true"

  • Result
  • HTML
  • Javascript 
<div id="result"
                    data-sgvizler-query="
                    SELECT (YEAR(?date) AS ?year) ?population WHERE {
                        wd:Q90  p:P1082 ?populationStatement.
                         ?populationStatement ps:P1082 ?population.
                         ?populationStatement pq:P585 ?date.
                       }
                       ORDER BY ?year"
                data-sgvizler-endpoint="https://query.wikidata.org/sparql"
                data-sgvizler-chart="google.visualization.BarChart"
                data-sgvizler-log="0">

></div>
<script src="../browser/sgvizler2.js"></script>

   //Draw a chart

   //or

   //Draw all Chart
